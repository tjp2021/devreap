# Code of Conduct

## Our Standard

We are committed to providing a welcoming and respectful environment for everyone, regardless of background or experience level.

## Expected Behavior

- Be respectful and constructive in discussions
- Accept feedback gracefully
- Focus on what's best for the project and community

## Unacceptable Behavior

- Personal attacks or inflammatory language
- Spam or off-topic disruption
- Publishing others' private information

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and other contributions that violate this code of conduct.

Report issues to **engineering@iteachyouai.com**. All reports will be reviewed and handled confidentially.

## Scope

This code of conduct applies to all project spaces — GitHub issues, pull requests, and discussions.
